Variables and calculations

Fields can be populated using experimental data from this dropdown menu or you can use your own custom data. Which ever you choose, you can always change the parameters and submit the new values to update the plots.

1. Environmental variables

In this section, the environmental variables are added to the model.

Changeable: Temperature C Salinity
Calculated: Oxygen concentration ... mg O2/L ... mg O2/m3
Calculated: Oxygen threshold ... % ... mg O2/m3
Updates: Plot B, C, D
Dissovled oxygen: >70% 70% - 40% 40% - 20% 20% - 0%

2. Swimming energetics

In this section the swimming energetics and critical swimmingspeed are added to calculate the optimale swimmingspeed in fed and unfed fish. Use the SDA field to add post prandial MO2. Results are plotted on the right

Changeable: SMR mg O2/kg/h Exp Ucrit BL/s
Changeable: SDA % of SMR
Calculated: Uopt - not fed ... BL/s Uopt - fed ... BL/s
Updates: Plot A, B, C, D

3. Cage dimensions

Here the cage dimensions are added in order to calculate the cage volume.

Changeable: Circumference m Depth m
Calculated: Diameter ... m Radius ... m Volume ... m3
Updates: Plot B, C, D

4. Fish density

In this section the total number of fish in the cage as well as the avg. mass and length are added, to calculate the total mass and density

Changeable: Number of fish x 1000 Avg. mass kg
Changeable: Avg. length cm
Calculated: Total mass ... kg Fish Density ... kg/m3
Updates: Plot B, C, D

5. Swimming and current information

In this section the swimming speed and current velocity are added for plot D.

Changeable: Water flow cm/s Swimming speed BL/s
Calculated: Water flow ... BL/s Swimmingspeed ... cm/s
Updates: Plot D
